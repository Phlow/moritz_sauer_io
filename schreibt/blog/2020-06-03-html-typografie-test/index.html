<s> – Durchgestrichener Text

Es ist super, wenn man Dinge erledigt bekommt. Streich einfach abgeschlossene Aufgaben durch.

<q> – Automatische Anführungsstriche

Wenn Du Text mit <q> umschließt, bekommt der eingeklammerte Text automatisch Anführungsstriche.

<dfn> – Definitionselement

Mit dem Element <dfn> wird ein Teil eines Fließtextes als Definition (definition) oder als zu definierenden Begriff ausgezeichnet.

<abbr> – Abkürzungen als Tooltip

Das Element <abbr> zeichnet einen Teil eines Fließtextes als Abkürzung, eng. abbreviation aus. Wenn die ausgeschriebene Langform im title-Attribut enthalten ist, wird sie bei :hover als Tooltip angezeigt. Beispiel: Der WWF ist eine der größten internationalen Naturschutzorganisationen der Welt.
