Hugo – Range-Schleifen

Bei meiner Webdesign-Arbeit mit dem Website Generator Hugo stolpere ich immer wieder über die Syntax von range-Schleifen. Hier eine Liste verschiedener Varianten, um Beiträge auszugeben.

Inhaltsverzeichnis

  1. Eine Section mit Index-Seite ausgeben
  2. Eine Section ohne Index-Seite ausgeben
  3. Ausgabe steuern nach Schlagworten
  4. Range und Data

Eine Section mit Index-Seite ausgeben

{{ range where .Site.Pages "Section" "blog" }}
  <li><a href="{{.Permalink}}">{{.Title}}</a></li>
{{ end }}

Eine Section ohne Index-Seite ausgeben

{{ range where .Site.RegularPages "Section" "blog" }}
  <li><a href="{{.Permalink}}">{{.Title}}</a></li>
{{ end }}

Ausgabe steuern nach Schlagworten

{{ range .Site.Taxonomies.tags.webdesign }}
  <li><a href="{{.Permalink}}">{{.Title}}</a></li>
{{ end }}

Range und Data

{{ range (index .Site.Data (.Get "data")) }}
{{ end }}
